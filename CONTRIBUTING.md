# Guia de Contribuição 🚀

Obrigado pelo seu interesse em contribuir para o projeto **Porto Máquinas**! Estamos felizes em ter você aqui.

## Como começar

1. Dê um *fork* neste repositório.
2. Crie uma branch para a sua feature ou correção de bug (`git checkout -b feature/minha-feature`).
3. Faça suas modificações com cuidado, garantindo que o estilo de código existente seja mantido (utilizamos o padrão de Glassmorphism e tema escuro).
4. Teste suas alterações localmente.
5. Faça o commit das suas mudanças (`git commit -m 'feat: Adiciona uma nova feature incrível'`).
6. Envie para o seu fork (`git push origin feature/minha-feature`).
7. Abra um Pull Request (PR) descrevendo bem as suas mudanças.

## Padrões de Código
- **HTML:** Utilize tags semânticas sempre que possível (`<main>`, `<article>`, `<section>`).
- **CSS:** As variáveis de cores globais estão no arquivo `colors.css`. Utilize-as em vez de "hardcodar" cores.
- **JS:** Comente funções complexas e garanta que os manipuladores de eventos não causem vazamentos de memória (memory leaks).

Se tiver dúvidas, abra uma Issue e ficaremos felizes em ajudar!
